## 📝 **CREAR NUEVOS PRESUPUESTOS:**

### **Template Emergent.sh:**
```
"Create a clean proposal presentation website for [CLIENT NAME]. 
Include:
- Header with company branding
- Project overview section  
- Pricing table (3 tiers: Discovery/Build/Ongoing)
- Timeline/deliverables list
- Contact section
- Modern, professional design
- Export as static HTML/CSS files
- No forms or interactions needed"
```

### **Proceso:**
1. **Emergent.sh** → Generar sitio para cliente
2. **Download** → Crear carpeta `/nuevo-cliente/`
3. **Replace** → Aplicar branding WTF desde `/shared/`
4. **Customize** → Ajustar pricing/timeline específico
5. **Git push** → Auto deploy Railway
6. **Share** → prospect.wtf-agency.com/nuevo-cliente/
